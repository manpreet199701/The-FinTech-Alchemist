Week 1: Introduction to Financial Data

In Week 1, I began by familiarizing myself with the basics of financial datasets. This involved learning how to use libraries like **Pandas** to load and clean raw stock data, calculate simple moving averages, and visualize price trends using **Matplotlib**. The goal was to understand the structure of financial data and how to perform fundamental analysis using code.

  • Learned to import and manipulate time-series data.
  • Calculated and plotted basic financial indicators.
  • Developed a script to download historical stock data.
